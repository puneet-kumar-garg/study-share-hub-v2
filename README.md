# Study Share Hub

A platform for sharing and learning together through educational worksheets.

## Project info

**Idea by**: Arunangshu Roy  
**Developed by**: Puneet Kumar Garg

## How to run this project locally

Follow these steps to set up the project on your local machine:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd study-share-hub

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Local Storage Authentication

## Features

- Simple local authentication system
- User dashboard
- Worksheet sharing and management
- Subject-based organization
- Responsive design

## Authentication

This project uses a simple local storage-based authentication system. No external services required - just create an account and start using the platform immediately.
